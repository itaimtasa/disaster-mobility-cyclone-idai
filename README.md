# Disaster Mobility After Cyclone Idai

A comparative mixed-methods analysis of post-disaster mobility in Mozambique and Zimbabwe.

Author: Itai Mtasa

Master's Thesis
Migration and Development
Sapienza University of Rome
2025

This repository contains the data, code, workflows, and research outputs used in my Master's thesis:

"Disaster Mobility: Post-Cyclone Idai Mobility in Mozambique and Zimbabwe – A Comparative Analysis of Countries and Mobility Actors"

The study investigates how interactions between governments, humanitarian actors, and communities shaped post-disaster mobility outcomes following Cyclone Idai (2019).

Using a mixed-methods approach, the research combines:

- IOM DTM displacement datasets
- News media analysis
- YouTube discourse analysis
- Humanitarian reporting analysis
- Sentiment analysis
- Comparative governance analysis

The objective is to understand mobility across a spectrum ranging from displacement and resettlement to return and immobility.

## Research Questions

Main Question:

How did interactions between national governance frameworks and humanitarian actor dynamics shape post-disaster mobility outcomes following Cyclone Idai in Mozambique and Zimbabwe?

Sub Questions:

1. How did formal mobility pathways differ between Mozambique and Zimbabwe?

2. To what extent do public media narratives align with community and humanitarian narratives?

3. How is immobility represented in public documentation versus observed mobility datasets?
